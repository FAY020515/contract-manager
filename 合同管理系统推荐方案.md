## 一、背景与现状

随着集团业务规模持续扩大，审核组承担的合同审核与管理工作量逐年增长。经梳理，当前合同管理存在以下突出问题：

| 问题 | 现状 | 影响 |
|------|------|------|
| 合同分散 | 纸质与电子文档混杂存放，无统一台账 | 查找耗时，易遗失 |
| 到期遗漏 | 依赖人工记忆或 Excel 手动标注提醒 | 合同到期未续签，存在业务中断风险 |
| 统计繁琐 | 每次汇总需手动翻表、逐条统计 | 报表产出慢，数据准确性难保证 |
| 协作困难 | 多人管理同一批合同，无统一平台 | 信息不对称，重复劳动 |
| 历史数据 | 往年合同散落在个人电脑或文件柜 | 交接困难，追溯成本高 |

目前市面上的商业合同管理软件（如泛微、用友、金蝶等）虽功能完善，但年费通常在 **数万元至数十万元** 不等，且部署周期长、定制化成本高，不太适合审核组当前的实际需求。

## 二、解决方案

本人利用业余时间，基于现代主流技术开发了一套**合同管理系统**，已在个人电脑上完成开发和测试，可有效解决上述问题。

### 核心功能

| 功能模块 | 说明 | 解决什么问题 |
|---------|------|------------|
| 合同录入 | 支持单条录入和 Excel 批量导入 | 快速建立电子台账，告别手工记录 |
| 合同查询 | 按编号、名称、类型、状态、部门等多条件组合检索 | 秒级定位目标合同，不再翻箱倒柜 |
| 到期提醒 | 系统自动在合同到期前 7/15/30 天创建提醒 | 杜绝遗漏，降低续签风险 |
| 附件管理 | 上传合同扫描件、附件，支持在线预览和下载 | 电子归档，纸质件不再是唯一凭证 |
| 统计分析 | 仪表盘、按类型/部门统计、月度趋势、金额分布图表 | 一键生成报表，数据可视化 |
| 用户管理 | 多用户支持，管理员/普通用户权限分离 | 各取所需，数据安全 |
| 数据备份 | 一键导出数据库备份文件 | 防止数据丢失，可定期存档 |

### 界面预览

系统采用现代化设计风格，界面简洁直观，上手无门槛：

- **仪表盘**：打开即见合同总量、执行中数量、即将到期预警、总金额等关键指标，一目了然
- **合同列表**：支持分页浏览、多条件筛选、关键词搜索，快速定位目标合同
- **统计图表**：柱状图、饼图、折线图等多种可视化展示，辅助领导决策
- **合同详情**：查看合同完整信息、操作日志、关联附件，历史记录可追溯

## 三、方案优势

### 与商业软件对比

| 对比项 | 商业软件（泛微/用友等） | 本系统 |
|--------|----------------------|--------|
| 采购成本 | 年费数万至数十万 | **零成本**，开源技术 |
| 部署周期 | 通常需数周至数月 | **当天安装，当天使用** |
| 定制能力 | 需联系厂商排期，费用另计 | **随时按需调整**，自主可控 |
| 数据安全 | 数据存储在厂商云端或指定服务器 | **数据完全存储在本地**，不经过任何外部服务器 |
| 运维依赖 | 需厂商技术支持或专职运维 | **无需专业运维**，出现问题可快速排查 |
| 使用门槛 | 功能繁多，培训成本高 | **界面简洁**，无需培训即可上手 |

### 核心亮点

- **部署灵活**：支持单机安装（双击即用）和局域网共享（多人浏览器访问）两种模式
- **数据安全**：所有数据存储在本地电脑，不依赖外部服务器，不经过任何云端，杜绝数据泄露风险
- **零成本**：全部采用开源技术栈，无需购买任何软件授权或许可证
- **可扩展**：架构清晰，后续可根据实际业务需求随时增加新功能

## 四、使用场景

### 场景一：日常合同管理
审核专员收到新签合同后，在系统中录入合同信息（或从 Excel 批量导入），上传合同扫描件作为电子归档，系统自动创建到期提醒。

### 场景二：批量建立历史台账
将往年存量合同按 Excel 模板整理后，一键批量导入系统，快速完成历史合同电子化，建立完整的合同台账。

### 场景三：领导查看统计报表
主任打开仪表盘，即可实时掌握合同总量、各部门合同分布、执行中合同总金额、即将到期合同等关键数据，无需等待人工汇总。

### 场景四：到期合同预警
系统自动在合同到期前 7 天、15 天、30 天生成提醒，审核专员打开系统即可看到待处理提醒，避免因遗忘导致的业务风险。

### 场景五：多人协作
将系统部署在部门一台电脑上，其他同事通过浏览器访问同一平台，各自录入和查看合同，数据实时同步，信息完全透明。

## 五、部署方案

### 方案一：单机版（推荐先试用）

适合个人或小范围试用，安装即用。

- 双击安装包，按提示完成安装
- 打开浏览器访问系统即可使用
- 数据自动保存在程序目录下的 `data` 文件夹中
- 关闭程序后数据不会丢失，下次打开自动恢复

### 方案二：局域网版（适合多人协作）

适合部门内多人共同使用。

- 在一台电脑上安装系统（作为"服务端"）
- 该电脑保持开机运行状态
- 其他同事在自己的电脑浏览器中输入指定地址即可访问
- 所有人共享同一份数据，实时同步

### 安装注意事项

- **安装路径**：建议安装到非系统盘（如 `D:\合同管理系统`），避免装在 `C:\Program Files` 下，确保数据文件可写
- **数据安全**：数据库文件与程序在同一目录，重启后自动保留，不受企业 IT 策略清理影响
- **自动迁移**：如从旧版升级，首次启动会自动检测并迁移历史数据，无需手动操作

### 硬件要求

- 操作系统：Windows 10 或 Windows 11
- 内存：4GB 以上
- 硬盘空间：200MB 安装空间 + 数据存储预留
