## 一、背景与需求

随着集团业务不断发展，审核组日常面临大量合同审核与管理工作，存在以下痛点：

- **合同分散**：纸质与电子文档混杂，查找困难
- **到期易忘**：合同到期提醒依赖人工，易遗漏
- **统计繁琐**：合同金额、类型、部门等数据汇总耗时
- **协作不便**：多人协作时缺乏统一平台，信息不对称

## 二、解决方案

本人利用业余时间，基于现代主流技术开发了一套**合同管理系统**，可有效解决上述问题。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 合同录入 | 支持单条录入和 Excel 批量导入 |
| 合同查询 | 按编号、名称、类型、状态、部门等多维度检索 |
| 到期提醒 | 自动创建到期提醒，支持系统通知推送 |
| 附件管理 | 上传合同扫描件、附件，支持在线预览和下载 |
| 统计分析 | 仪表盘、按类型/部门统计、月度趋势、金额分布图表 |
| 用户管理 | 多用户支持，管理员/普通用户权限分离 |
| 数据备份 | 一键导出数据库备份，保障数据安全 |

### 界面预览

系统采用现代化设计，界面简洁直观：

- **仪表盘**：一目了然掌握合同总量、执行中、即将到期、总金额等关键指标
- **合同列表**：支持分页、筛选、搜索，快速定位目标合同
- **统计图表**：柱状图、饼图、折线图等多种可视化展示

## 三、技术优势

| 特性 | 说明 |
|------|------|
| 部署灵活 | 支持桌面应用（安装即用）和网页版（局域网访问）两种模式 |
| 数据安全 | 数据存储在本地，不依赖外部服务器，避免数据泄露风险 |
| 零成本 | 使用开源技术，无需购买商业软件授权 |
| 易维护 | 无需专业运维人员，出现问题可快速排查解决 |
| 可扩展 | 架构清晰，后续可根据需求定制开发新功能 |

## 四、使用场景

### 场景一：日常合同管理
审核专员录入新签合同，上传扫描件，设置到期提醒。

### 场景二：批量导入历史合同
通过 Excel 模板批量导入存量合同数据，快速建立合同台账。

### 场景三：领导查看统计报表
主任可通过仪表盘和统计页面，实时掌握部门合同整体情况。

### 场景四：到期合同预警
系统自动提醒即将到期的合同，避免因遗忘导致的业务风险。

## 五、部署方案

### 方案一：单机版（推荐试用）
- 安装包直接运行，无需服务器
- 数据存储在本地电脑
- 适合个人或小范围试用

### 方案二：局域网版
- 部署在部门内一台电脑上
- 其他同事通过浏览器访问
- 适合部门内多人协作使用

### 硬件要求
- 操作系统：Windows 10/11
- 内存：4GB 以上
- 硬盘：100MB 安装空间 + 数据存储空间
